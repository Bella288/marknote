Markdown Cheatsheet

Markdown Quick Reference

  • Headers: # H1, ## H2, ### H3
  • Emphasis: *italic*, **bold**, ~~strikethrough~~
  • List: - item 1, - item 2 or 1. item 1, 2. item 2
  • Links: [Text](URL)
  • Images: ![Alt text](Image URL)
  • Code: `inline code` or ```code block```
  • Blockquotes: > Quoted text
  • Horizontal rule: ---
  • See the full: Markdown Cheatsheet for more examples
